# React + TypeScript + Vite Frontend

This frontend is a minimal setup using Vite with React and TypeScript. It includes:

- Fast refresh enabled by the official Vite React plugin (`@vitejs/plugin-react`).
- ESLint configured for React with TypeScript for improved code quality.
- React Query and Axios for data fetching from the backend.
