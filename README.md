# Spring Boot Demo Project

A simple Spring Boot application demonstrating basic REST API endpoints.

## Features

- REST API endpoints
- JSON response handling
- Path variables and request body processing
- Basic application information endpoint

## Prerequisites

- Java 17 or higher
- Maven 3.6 or higher
