# Simple Spring Boot Application

A simple Spring Boot application built with Gradle.

## Features

- Spring Boot 3.2.0.
- Spring Data JPA.
- H2 Database.
- RESTful API endpoints.
- User management functionality.
- Actuator for monitoring.

## Prerequisites

- Java 17 or higher.
- Gradle (or use the included wrapper).
